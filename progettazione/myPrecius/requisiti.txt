REQUISITI

-Opera, si vuole sapere:
    -nome: Stringa
    -categoria: Categoria
    -autore (vedere Autore)
    -anno_realizzazione: Intero > 0
    -tecnica (se rilevante): Tecnica [0..1]
    -correnti_artistiche: Corrente

-Autore, si vuole sapere:
    -nome_artistico: Stringa
    -nascita: Data
    -morte (se morto): Data [0..1]
    -opere (vedere Opera)

-Città, si vuole sapere:
    -nome: Stringa

-Regione, si vuole sapere:
    -nome: Stringa

-Nazione, si vuole sapere:
    -nome: Stringa

-Esposizione, può essere (Permanente, Temporanea)

-Permanente

-Temporanea, si vuole sapere:
    -nome: Stringa
    -intervallo_date: Data [1..2]
    -prezzo: Reale > 0 (FORSE probabilmente removibile con link Tariffa)

-Tariffa, si vuole sapere:
    -nome: Stringa
    -prezzo_base: Reale >= 0


-Biglietto, si vuole sapere, può essere(Standard, ExtendedAccess):

    -data_validità: Data (OPPURE DataOra OPPURE operazione calcolabile ? scelto dal visitatore, association class ?)
